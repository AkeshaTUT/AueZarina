# Переменные окружения для Replit
# Добавьте эти переменные в Secrets панели Replit

# 1. TELEGRAM_BOT_TOKEN - ваш токен от @BotFather
# 2. OPENROUTER_API_KEY - ваш ключ для ИИ (необязательно)
# 3. STEAM_WEB_API_KEY - ваш Steam API ключ (необязательно)

# Инструкция:
# 1. В Replit перейдите в панель Secrets (значок замка в левом меню)
# 2. Добавьте следующие секреты:
#    - Key: TELEGRAM_BOT_TOKEN, Value: ваш_токен_бота
#    - Key: OPENROUTER_API_KEY, Value: ваш_ключ_openrouter
#    - Key: STEAM_WEB_API_KEY, Value: ваш_steam_ключ
